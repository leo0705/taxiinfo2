Пояснения и рекомендации по использованию

1 Кнопка посылает запрос на поиск и переключает видимость на панель с результатами запроса.
Использование: кликнуть для просмотра результатов выполнения запроса.

2 Кнопка      управляет списком для выбора значения параметров.
Использование: кликнуть, выпадающее окно закроется (если оно открыто), либо откроется содержащее список значений параметра, попадающих в результат запроса (не более 100).
Если Вы желаете изменить уже введенное значение, то предварительно поле надо очистить (т.е. сбросить старое значение с использованием кнопки      ).
Открытое выпадающее окно закрывается в трех случаях:

  • после выбора значения из списка;
  • после клика по данной кнопке, используемой для этого либо любого другого параметра;
  • после клика по кнопке "загрузка";

3 Кнопка позволяет выбрать из списка требуемое значение.
Использование: в выпадающем окне кликнуть кнопку перед требуемым значением, выпадающее окно закроется.

4 Кнопка      позволяет сбросить введенное значения параметра запроса.
Использование: кликнуть, если поле содержит значение, а Вы не хотите включать в запрос этот параметр.

5 Кнопка      позволяет открыть поле для ввода c клавиатуры.
Использование: кликнуть, затем вводить с клавиатуры требуемое значение, начиная с его первого символа (буквы русские).
В выпадающем окне выводится список возможных значений, первые символы которых совпадают с указанными Вами.
В любой момент Вы можете прекратить ввод и кликнуть кнопку "загрузить из БД". При этом если в поле ввода остались только символы, введенные Вами с клавиатуры, то в результат запроса будет включено все, что было в выпадающем окне. Если же Вы перед тем, как кликнуть кнопку "загрузка", выбрали из выпадающего окна конкретное значение, то в результат запроса будет включено только оно.

6 Кнопка позволяет при просмотре результатов поиска перейти на панель для ввода параметров поиска.
Использование: кликнуть для перехода к настройке параметров поиска в БД.

7 Кнопка      позволяет вернуться к началу страницы.
Использование: кликнуть для перехода к началу страницы.

Формирование результата запроса

Набор введенных значений в поля параметров запроса является фильтром, который пропускает в результат только те строки, которые удовлетворяют всем введенным значениям.
После каждого изменения значения любого параметра пересчитывается и выводится количество записей БД, которые попадают в результат запроса.
Если при вводе значения параметра Вы получаете сообщение "не загружено ни одной записи", то это означает, что в БД не существует записи, которая удовлетворяет всем указанным Вами условиям.

Содержимое списка, которое выводится в выпадающем окне при вводе каждого из параметров, формируется по точно такому же алгоритму, как и формирование окончательного результата, т.е. с учетом всех установленных значений для других параметров (но не более 100 записей для полей "Регистрационный номер" и "Перевозчик", точное число показывается в поле для вывода сообщений).
