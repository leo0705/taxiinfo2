Пояснения по вводу параметров запроса
  • 1 - Кнопка "загрузка" посылает запрос на поиск и переключает видимость на панель с результатами запроса.
    Использование: кликнуть для просмотра результатов выполнения запроса.
  • 2 - Поля для ввода параметров "Марка" и "Модель". Содержат кнопку, которая открывает и закрывает выпадающее окно, содержащее список возможных значений параметра запроса.
  • 3 - Поля "Номерной знак" и "Перевозчик" позволяют использовать для ввода и мышь и клавиатуру.
    Использование: кликнуть внутри поля, выпадающее окно закроется (если оно открыто), либо откроется (если оно закрыто) содержащее список значений параметра, попадающих в результат запроса(не более 100).
    Использование: Вводить с клавиатуры требуемое значение, начиная с его первого символа (буквы русские). В выпадающем окне выводится список возможных значений, первые символы которых совпадают с указанными Вами.
    Использование: в выпадающем окне кликнуть требуемое значение, выпадающее окно закроется.
    В любой момент Вы можете прекратить ввод и кликнуть кнопку "загрузка". При этом если в поле ввода остались только символы, введенные Вами с клавиатуры, то в результат запроса будут включено все, что было в выпадающем окне. Если же Вы перед тем, как кликнуть кнопку "загрузка", выбрали из выпадающего окна конкретное значение, то в результат запроса будет включено только оно.
  • 4 - Кнопка, которая открывает и закрывает выпадающее окно, содержащее список возможных значений параметра запроса.
    Если Вы желаете изменить уже введенное значение, то предварительно поле надо очистить (сбросить старое значение).
    Использование: кликнуть кнопку и в выпадающем окне кликнуть требуемое значение.
    Открытое выпадающее окно закрывается в трех случаях:
    • после выбора значения из списка;
    • после клика по кнопке, даже если эта кнопка используется для ввода значения другого параметра;
    • после клика по кнопке "загрузка";
  • 5 - Кнопка для сброса введенного значения параметра запроса.
    Использование: кликнуть, если поле содержит значение, а Вы не хотите включать в запрос этот параметр.
  • 6 - Поле для вывода сообщений о выполнении запроса. После каждого изменения значения любого параметра пересчитывается и выводится количество записей БД, которые попадают в результат запроса.
    Использование: кликнуть в поле, если Вам захочется его очистить.

Формирование результата запроса

  • Набор введенных значений в поля параметров запроса является фильтром, который пропускает в результат только те строки, которые удовлетворяют всем введенным значениям.
  • Содержимое списка, которое выводится в выпадающем окне при вводе каждого из параметров, формируется по точно такому же алгоритму, т.е. с учетом всех установленных значений для других параметров(но не более 100 записей для полей "Номерной знак" и "Перевозчик", точное число показывается в поле для вывода сообщений).
  • Когда Вы при вводе значения параметра получаете сообщение "не загружено ни одной записи", то это означает, что в БД не существует записи, которая удовлетворяет всем указанным Вами значениям.
